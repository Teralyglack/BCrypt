# Цель проекта

Разработать консольное приложение на C# с использованием библиотеки BCrypt.Net-Next для демонстрации безопасного хэширования паролей. Приложение должно служить учебным примером для понимания принципов хранения паролей в базах данных.

## Задачи

1. Реализовать ввод пароля пользователем из консоли
2. Реализовать хэширование пароля с использованием BCrypt.Net-Next
3. Реализовать повторный ввод пароля для верификации
4. Вывести результат сравнения в консоль
5. Добавить обработку ошибок (пустой пароль, null значение)
6. Создать README.md с документацией по использованию

## Функциональные требования

1. Приложение должно запрашивать у пользователя пароль
2. Приложение должно генерировать хэш введенного пароля
3. Приложение должно запрашивать пароль повторно
4. Приложение должно сравнивать повторно введенный пароль с хэшем первого
5. Приложение должно выводить сообщение о совпадении или несовпадении паролей
6. Приложение должно обрабатывать пустой ввод и выводить сообщение об ошибке
7. Хэш должен включать соль и work factor (стоимость)

## Нефункциональные требования

1. Код должен быть простым и понятным для начинающих разработчиков
2. Приложение должно работать в среде Windows, Linux, macOS
3. Время хэширования не должно превышать 1 секунды (work factor = 10)
4. Библиотека BCrypt.Net-Next должна быть установлена через NuGet
5. Проект должен быть совместим с .NET 6.0 или выше
6. Код должен содержать минимальное количество зависимостей

## Ограничения

1. Приложение не сохраняет пароли и хэши в файлы или базу данных
2. Приложение не передает данные по сети
3. Work factor фиксирован (10) и не настраивается пользователем
4. Приложение завершается после одной проверки (без цикла)
5. Не поддерживается графический интерфейс, только консоль
6. Пароль не маскируется звездочками при вводе
