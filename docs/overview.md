## Используемые компоненты

### Основные компоненты

| Компонент | Версия | Назначение |
|-----------|--------|------------|
| .NET | 6.0 / 8.0 | Платформа выполнения |
| C# | 10.0 / 12.0 | Язык программирования |
| BCrypt.Net-Next | 4.1.0 | Библиотека хэширования |
| Console | - | Интерфейс ввода-вывода |

### Системные компоненты

- System.Console - работа с консолью
- System.Exception - обработка ошибок
- RNGCryptoServiceProvider - генерация соли (внутри библиотеки)

### Внешние зависимости

- BCrypt.Net-Next (NuGet пакет)
- .NET Runtime

## Что реализовано в проекте

### Реализованный функционал

1. **Ввод пароля**
   - Чтение строки из консоли
   - Проверка на пустой ввод
   - Проверка на null значение

2. **Хэширование пароля**
   - Использование BCrypt.Net-Next
   - Автоматическая генерация соли
   - Work factor = 10 (1024 раунда)
   - Формат вывода: $2a$10$...

3. **Верификация пароля**
   - Повторный ввод
   - Сравнение с хэшем
   - Вывод результата

4. **Обработка ошибок**
   - Пустой пароль
   - Исключения BCrypt
   - Общий блок try-catch
